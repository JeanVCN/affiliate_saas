# Roadmap

## MVP

Goal: validate that affiliates pay for organization, campaign generation, link tracking, and conversion imports.

Scope:

- Auth and workspaces.
- Manual marketplace/program setup.
- Product and affiliate link registry.
- Initial playbooks: TikTok Shop, Shopee, Mercado Livre, Amazon.
- Short links and click tracking.
- AI Campaign Studio.
- Channel templates for TikTok, Reels, Shorts, WhatsApp, Telegram, Blog.
- TikTok Shop templates: UGC review, achadinhos, comparison, demonstration, flash offer.
- Simple calendar.
- CSV/manual sales and commission imports.
- Dashboard for clicks, imported sales, commission, top products.
- Basic compliance checklist.

## Version 1.0

- Amazon integration if approved.
- TikTok Content Posting API if approved.
- Dedicated TikTok Shop affiliate report importer after format validation.
- CSV importers for Shopee, Mercado Livre, AliExpress, Magalu.
- Simple landing pages.
- Product comparison builder.
- Creative library.
- Channel reports.
- Small-team workflows.
- Zapier/Make/webhooks.

## Version 2.0

- Approved official marketplace integrations.
- Allowed price/offer monitoring.
- Product recommendation by category.
- A/B testing for copy and links.
- Official social publishing or partner integrations.
- Agency portal.
- Basic white label.

## Version 3.0

- Campaign marketplace.
- White-label affiliate portal for brands.
- Performance data warehouse.
- Predicted EPC by product/channel.
- Public API.
- Enterprise SSO.
