# Product Vision

Affiliate SaaS is an AI-assisted operating system for affiliate commerce.

The product helps creators, affiliate operators, agencies, and small businesses manage products, affiliate links, campaign copy, publishing tasks, click tracking, sales imports, and performance insights across multiple affiliate programs.

## Core Thesis

Do not compete head-on with generic video clipping tools. Start with the more defensible commerce workflow:

`product -> affiliate link -> campaign assets -> channel-specific copy -> tracking -> imported conversions -> insight -> next campaign`

## Initial Platforms

- TikTok Shop Affiliate.
- Shopee Affiliates.
- Mercado Livre affiliate/parceiro workflows.
- Amazon Associates/Creators.

The MVP must work without closed marketplace API access by supporting manual entry, link management, CSV imports, and user-authorized reports.

## Non-Goals For MVP

- Scraping marketplaces or social networks.
- Browser automation for posting or engagement.
- Automated likes, follows, comments, views, or DMs.
- Heavy video editor.
- Full marketplace API dependency.

## Success Signal

The product is working when a user can answer:

- Which product should I promote next?
- What copy should I use for each channel?
- Which link/campaign generated clicks and conversions?
- Which product/channel/angle should I repeat?
